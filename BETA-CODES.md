# Codes d'invitation Bêta Intralys

Ces codes peuvent être distribués aux 5 premiers clients. Ils permettent de s'inscrire gratuitement.

1. `BETA-INTRALYS-2026-X7K9`
2. `BETA-INTRALYS-2026-M4P2`
3. `BETA-INTRALYS-2026-L8V5`
4. `BETA-INTRALYS-2026-R3N1`
5. `BETA-INTRALYS-2026-Q9J4`

## Fonctionnement

Lors de la création de la page d'inscription publique (ou si distribués manuellement), ces codes devront être validés dans la table `beta_invite_codes`.
Pour le MVP, le simple fait d'avoir un code valide autorisera la création du compte.

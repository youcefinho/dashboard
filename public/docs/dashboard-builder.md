# Le dashboard builder

Le tableau de bord par défaut ne te suffit pas ? Construis le tien, bloc par bloc.

## Ouvrir le builder

Va dans **Rapports → Mes tableaux de bord → Nouveau**. Tu pars d'une grille vide ou d'un preset.

## Ajouter des blocs

Clique **Ajouter un bloc** et choisis un type :

- **KPI** — un chiffre clé (leads, conversion, CA)
- **Graphique** — courbe, barres, camembert
- **Liste** — top leads, tâches, activité
- **Table** — données détaillées filtrables

## Configurer un bloc

Chaque bloc se paramètre : métrique, période, filtre (source, propriétaire, pipeline), regroupement. Aperçu en direct.

## Disposition

Glisse-dépose les blocs pour réorganiser. Redimensionne-les (1/3, 1/2, pleine largeur). La grille s'adapte au mobile automatiquement.

## Sauvegarder & partager

Nomme ton dashboard et enregistre. Tu peux le **définir par défaut**, ou le partager à l'équipe (selon les rôles). Plusieurs dashboards = un par usage (vente, marketing, direction).

## Exporter

Tout dashboard s'exporte en **PDF** propre (mise en page A4) pour une réunion ou un rapport client — voir [Export PDF](/help/export-pdf).

# Réactions

Une **réaction** est un emoji collé sur un message. Le moyen le plus rapide de dire « reçu » ou « 👍 ».

## Réagir à un message

Dans une conversation, survole le message reçu (ou appui long sur mobile). Une **barre d'emojis** apparaît : 👍 ❤️ 😂 🎉 👀 ✅. Clique pour réagir.

## Côté contact

Selon le canal, la réaction est visible par le contact (comme sur WhatsApp/iMessage). C'est un signal léger : « j'ai vu, je traite ».

## Persistance

Les réactions sont **sauvegardées** sur le message et restent visibles dans la timeline du lead. Tu peux retirer une réaction en cliquant à nouveau dessus.

## Quand utiliser une réaction plutôt qu'un message

- ✅ Accuser réception sans lancer un échange
- 👍 Valider une proposition rapidement
- 👀 Signaler « je regarde, ça vient »

Pour tout ce qui demande une réponse réelle, écris un message (ou utilise l'[IA de rédaction](/help/ia-redaction)).

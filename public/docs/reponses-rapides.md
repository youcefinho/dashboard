# Réponses rapides

Tu réécris souvent les mêmes réponses ? Les **réponses rapides** te font gagner ce temps.

## Le principe

Une réponse rapide = un bout de texte préenregistré (avec slash-variables possibles) que tu insères en un clic dans la conversation.

## Réponses rapides par lead

Au-dessus de la zone de saisie, Intralys propose des **chips de réponses rapides contextuelles** : les 3 dernières réponses utiles pour ce lead, en FIFO (la plus ancienne sort quand une nouvelle entre). Un clic = inséré.

## Créer une réponse réutilisable

Pour une réponse type permanente (« nos tarifs », « nos heures d'ouverture »), crée un **template** dans **Settings → Templates**. Donne-lui un nom court pour le retrouver vite.

## Bonnes réponses rapides

- Courtes et adaptables (laisse la place à une phrase perso)
- Avec slash-variables pour rester personnel (`/prenom`)
- Une intention claire par réponse (ne mélange pas tarifs + RDV)

## Réactions emoji

Pour accuser réception sans rédiger, survole un message reçu → barre d'emojis → réagis. Le contact voit ta réaction (selon le canal). Pratique pour un « 👍 reçu, je regarde ».

# Tags & segmentation

Les **tags** sont des étiquettes libres que tu colles sur tes leads pour les regrouper. La **segmentation**, c'est s'en servir pour cibler.

## Créer et appliquer un tag

Ouvre un lead → champ **Tags** → tape un nom (ex. `vip`, `relance-printemps`, `clinique-laval`). S'il existe, il s'autocomplète ; sinon il se crée.

Tu peux aussi taguer **en lot** : sélectionne plusieurs leads dans la liste → barre d'action → Ajouter un tag.

## Bonnes pratiques de nommage

- Reste cohérent : `secteur-courtage`, pas `Courtage` ET `courtier`
- Préfixe par catégorie : `src-fb`, `src-google`, `camp-noel`
- Évite les tags jetables qui ne resserviront jamais

## Segmenter avec les listes intelligentes

Une **liste intelligente** = un filtre sauvegardé qui se met à jour seul. Exemples utiles :

- « VIP sans contact depuis 14 jours »
- « Source Facebook + score ≥ 60 »
- « Tag relance-printemps + statut Nouveau »

Crée-la depuis la page Leads : applique tes filtres → **Enregistrer comme liste**. Elle apparaît dans la barre latérale.

## Tags vs statuts vs étapes

- **Statut** : où en est le lead globalement (Nouveau, Actif, Perdu…)
- **Étape pipeline** : position dans ton tunnel de vente
- **Tag** : tout le reste (campagne, secteur, préférence) — flexible et cumulable

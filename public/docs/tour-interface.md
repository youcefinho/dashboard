# Tour de l'interface

Intralys est conçu pour aller vite. Voici les repères essentiels.

## La barre latérale (gauche)

C'est ton menu principal. De haut en bas :

- **Tableau de bord** — vue d'ensemble, tes chiffres du jour
- **Leads** — la liste de tous tes prospects
- **Pipeline** — vue Kanban de tes deals par étape
- **Tâches** — ce que tu dois faire (rappels, suivis)
- **Inbox** — messagerie unifiée (courriel, SMS, WhatsApp)
- **Calendrier** — rendez-vous et disponibilités
- **Rapports** — analyses et tableaux de bord personnalisés
- **Settings** — paramètres équipe, branding, intégrations

Chaque item peut afficher une **pastille colorée** : nombre de nouveaux leads, tâches en retard, messages non lus.

## La palette de commandes

Tape `Cmd+K` (Mac) ou `Ctrl+K` (Windows) n'importe où. C'est le moyen le plus rapide de :

- Naviguer vers une page
- Chercher un lead
- Lancer une action (créer lead, nouvelle tâche)
- Filtrer une liste

## Raccourcis clavier utiles

| Touche | Action |
|--------|--------|
| `Cmd/Ctrl+K` | Palette de commandes |
| `C` | Créer un lead (Quick-Add) |
| `G` puis `L` | Aller aux Leads |
| `G` puis `P` | Aller au Pipeline |
| `Shift+R` | Rafraîchir les données |
| `?` | Afficher tous les raccourcis |

## Le bouton + flottant

En bas à droite, toujours visible. Un clic déploie les actions rapides : nouveau lead, nouvelle tâche, nouveau message.

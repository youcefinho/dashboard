# Créer un lead manuellement

## Étape 1 — Nouveau lead

Depuis **Leads**, cliquez sur **+ Nouveau lead**.

## Étape 2 — Renseignez les informations

- Nom (obligatoire)
- Email
- Téléphone
- Source (formulaire web, Facebook, référence, etc.)
- Étape du pipeline

## Étape 3 — Ajoutez des tags

Les tags vous permettent de segmenter vos leads (ex: "VIP", "Nouveau client").

## Étape 4 — Assignez à un membre

Sélectionnez le responsable du suivi pour ce lead.

## Étape 5 — Sauvegardez

Le lead est créé. Vous pouvez immédiatement lui envoyer un message, créer une tâche ou planifier un RDV.

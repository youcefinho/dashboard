# Créer votre premier workflow

Les workflows vous permettent d'automatiser vos tâches répétitives.

## Exemple : Message de bienvenue

1. Allez dans **Automatisations** > **Workflows**.
2. Cliquez sur **Nouveau Workflow**.
3. Choisissez le déclencheur : `Nouveau Lead`.
4. Ajoutez une action : `Envoyer un Courriel`.
5. Rédigez votre courriel de bienvenue.
6. Enregistrez et activez le workflow en haut à droite.

Désormais, chaque nouveau lead recevra automatiquement ce courriel !

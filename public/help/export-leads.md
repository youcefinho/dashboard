# Exporter vos leads

## Export CSV complet

Depuis **Leads**, cliquez sur **Exporter → CSV**. Tous les leads visibles selon vos filtres seront exportés.

## Colonnes incluses

Par défaut, l'export inclut :
- Nom, Email, Téléphone
- Statut, Tags
- Source, Date de création
- Responsable
- Montant estimé

## Personnaliser les colonnes

Avant l'export, cliquez sur **Personnaliser** pour choisir les champs à inclure.

## Export filtré

Appliquez vos filtres habituels avant de cliquer sur Exporter. Seuls les leads correspondants seront dans le fichier.

## Conformité Loi 25

Si vous exportez des données personnelles, vous êtes responsable de leur traitement conforme à la Loi 25 du Québec.

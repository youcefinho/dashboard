# Comment importer vos leads (CSV)

Vous pouvez facilement importer vos contacts existants dans Intralys via un fichier CSV.

## Étapes

1. Allez dans l'onglet **Leads**.
2. Cliquez sur le bouton **Importer**.
3. Sélectionnez votre fichier `.csv`.
4. Mappez les colonnes de votre fichier avec les champs Intralys (Nom, Email, Téléphone).
5. Confirmez l'import.

> **Astuce** : Vous pouvez télécharger notre [modèle CSV](#) pour vous assurer que vos données sont au bon format.

# Automatiser avec Zapier

Connectez Intralys à 5000+ applications via Zapier.

## Étape 1 — Compte Zapier

Créez un compte gratuit sur zapier.com.

## Étape 2 — Trouver Intralys

Dans Zapier, cherchez "Intralys" comme app de déclencheur ou d'action.

## Étape 3 — Connexion

Vous aurez besoin de votre clé API Intralys, disponible dans **Paramètres → API → Générer une clé**.

## Déclencheurs disponibles

- Nouveau lead créé
- Lead change de statut
- Nouvelle conversation
- Paiement reçu
- RDV réservé

## Actions disponibles

- Créer un lead
- Ajouter un tag
- Envoyer un message
- Créer une tâche
- Mettre à jour un champ

## Exemples populaires

- Formulaire Typeform → créer lead Intralys
- Stripe payment → créer facture Intralys + tag "Client payant"
- Nouveau lead Intralys → ligne Google Sheets

## Coût

Plan Zapier gratuit : 100 tâches/mois. Plan Starter : 750 tâches/mois (~30$/mois).

# Notes et activités sur un lead

Toute interaction avec un lead est tracée dans la timeline.

## Ajouter une note

Sur la fiche lead, onglet **Activités**, cliquez sur **+ Note**. Idéal pour documenter une conversation téléphonique ou un détail important.

## Types d'activités

- **Note manuelle** — Texte libre
- **Email envoyé/reçu** — Automatique
- **SMS envoyé/reçu** — Automatique
- **Appel** — Manuel ou automatique selon intégration
- **Tâche complétée** — Automatique
- **Changement de statut** — Automatique

## Mentions @

Mentionnez un collègue avec **@nom** dans une note pour le notifier directement.

## Recherche dans les notes

Utilisez la recherche globale en haut de l'écran pour retrouver une note par mot-clé.

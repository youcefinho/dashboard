# Gérer le pipeline Kanban

La vue Pipeline affiche vos leads sous forme de colonnes Kanban, organisés par statut.

## Glisser-déposer

Faites glisser une carte lead d'une colonne à l'autre pour changer son statut. Le changement est sauvegardé instantanément.

## Vue compacte vs détaillée

Basculez entre les modes :
- **Compact** — Plus de leads visibles
- **Détaillé** — Plus d'infos par carte (montant, dernière activité, responsable)

## Filtres rapides

Filtrez par :
- Responsable
- Tags
- Date de création
- Montant min/max

## Tri intelligent

Par défaut, les leads sont triés par dernière activité. Vous pouvez aussi trier par montant, date de création ou priorité.

## Vue Liste

Si vous préférez le format tableau, basculez sur **Vue Liste** depuis le menu en haut à droite.

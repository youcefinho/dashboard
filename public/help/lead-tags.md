# Organiser avec les tags

Les tags vous permettent de classer vos leads de manière flexible.

## Ajouter un tag

Sur une fiche lead, cliquez sur **+ Tag** et tapez le nom du tag. Les tags existants sont suggérés automatiquement.

## Filtrer par tag

Sur la page Leads, cliquez sur **Filtres → Tags** pour afficher uniquement les leads correspondant à un ou plusieurs tags.

## Couleurs de tags

Dans **Paramètres → Tags**, assignez une couleur à chaque tag pour une identification visuelle rapide.

## Tags vs Pipelines

- **Pipeline** : où en est le lead dans le processus de vente
- **Tags** : caractéristiques transversales (priorité, source, segment, etc.)

## Bonnes pratiques

Limitez-vous à 10-15 tags principaux. Trop de tags rendent la segmentation confuse.

# Utiliser Intralys sur mobile

Intralys est conçu mobile-first. Toutes les fonctionnalités sont disponibles depuis votre téléphone.

## Installation comme app (PWA)

1. Ouvrez **crm.intralys.com** dans Safari (iOS) ou Chrome (Android).
2. Tapez le bouton **Partager** puis **Ajouter à l'écran d'accueil**.
3. L'icône Intralys apparaît comme une vraie application.

## Navigation mobile

La barre du bas vous donne accès rapidement à : Tableau de bord, Leads, Conversations, Calendrier et Plus.

## Mode "Agent en visite"

Lorsque vous êtes en déplacement, activez le mode visite depuis la page Lead pour prendre des photos, ajouter des notes vocales et enregistrer l'emplacement GPS.

## Notifications push

Activez les notifications dans **Paramètres → Notifications** pour ne rater aucun nouveau lead ou message entrant.

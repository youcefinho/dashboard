Site Démo du Courtier

Ceci est une page de test pour valider l'intégration du widget Webchat Intralys.

Comment tester ?

  1. Vérifiez qu'un bouton flottant apparaît en bas à droite de cette page.
  2. Cliquez sur le bouton pour ouvrir l'interface de chat.
  3. Remplissez le formulaire de pré-chat (Nom, Email).
  4. Envoyez un message.
  5. Dans un autre onglet, ouvrez le Dashboard CRM Intralys (http://localhost:5174/inbox).
  6. Vous devriez voir la nouvelle conversation apparaître. Répondez depuis le CRM.
  7. Le message devrait apparaître ici en temps réel !
